# HR Analytics Dashboard (Tableau)

## Overview

This project presents an interactive **HR Analytics Dashboard** created using **Tableau**. It visualizes key workforce metrics to help HR professionals monitor employee trends, analyze attrition, and make data-driven decisions. The dashboard aims to deliver actionable insights into the human resources landscape of a company using clean and intuitive visualizations.

## Objectives

- Analyze employee demographics and distribution  
- Understand attrition patterns across departments and roles  
- Identify hiring trends and workforce diversity  
- Assist HR teams in strategic decision-making  

## Dashboard Features

- **Total Employees & Department-Wise Breakdown**  
- **Gender & Education Distribution**  
- **Attrition Analysis by Department, Gender, Age, and Education**  
- **Hiring Trend Over Time**  
- **Average Monthly Income by Role/Department**  
- Interactive filters for dynamic exploration  

## Tools & Technologies

- **Tableau** – For data visualization  
- **Microsoft Excel** – For data cleaning and preparation  

## Dataset

- **Source**: Sample HR dataset (anonymized)  
- **Fields Included**: Employee ID, Department, Gender, Age, Attrition, Education Field, Monthly Income, Date of Joining, etc.

1. Download the `.twbx` file in this repository
2. Open it using Tableau Desktop (Public or Professional version)
3. Interact with filters and charts to explore insights
